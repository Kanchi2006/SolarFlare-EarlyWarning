# SolarGuard AI

## Overview

SolarGuard AI is an Aditya-L1 inspired multi-agent solar flare forecasting system that analyzes solar X-ray intensity and predicts potential impacts on GPS, satellites, and communication systems.

## Problem Statement

Solar flares can disrupt:

* GPS navigation
* Satellite communications
* Aviation and maritime operations
* Electrical power grids

Timely detection and interpretation of solar activity are critical for reducing operational risks.

## Solution

SolarGuard AI uses a multi-agent architecture to:

* Validate solar observations
* Predict flare risk levels
* Generate alerts
* Explain impacts using Gemini AI
* Recommend mitigation actions
* Generate downloadable early-warning reports

## Multi-Agent Architecture

1. Security Agent
2. Prediction Agent
3. Alert Agent
4. Gemini Explanation Agent
5. Recommendation Agent
6. Report Agent
